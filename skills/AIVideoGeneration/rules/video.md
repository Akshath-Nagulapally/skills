# Workflow Summary:

In order to create an AI-Generated video and integrate it into our project, you will follow the steps in the order shown below:

1) Brainstorm prompt (Based on user's idea)
2) Endpoint Calling
3) Download
4) Integration

# Brainstorm Prompt:

You will refine the user's idea further in order to make the video generated by the video generation model in the next step better.

# Calling Endpoint

The video generation runs locally, as a result, here is how to call it:

```bash
curl -X POST http://localhost:8080/generate-video \
  -H "Content-Type: application/json" \
  -d '{"prompt": your_prompt }
```

# Download and Integration

The output of the previous step comes in the form of a url string. In order to integrate it into our remotion project:

1. Download and save into the public directory in the sandbox:

```bash
wget "URL" -O public/my_custom_name.mp4
```

2. Integrate it into the remotion project: Based on the plan that the user and you have already discussed
